## Тестовая среда

**Clickhouse 21.3.20**.

Сборка:
````bash
docker-compose up -d
````

Инициализация данных:
````bash
docker-compose exec clickhouse ./init.sh
````

Запуск теста:
````bash
docker-compose exec clickhouse ./test.sh
````

Очистка:
````bash
docker-compose down -v
````
